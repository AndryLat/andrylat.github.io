Java Core

Теория Java Core Теория нужна. На этом я мог бы и остановиться, но все же стоит раскрыть этот пункт. Теория нужна и ей необходимо качественно оперировать. Это значит, что вам обязательно нужно знать весь Java Core, понимать (!) устройство Java хотя бы поверхностно и знать всю основную функциональность стандартной библиотеки. Но важно не зарываться в самые дебри, которые будут требовать много времени и не дадут вам результата. Без примеров тут не обойтись.

Roadmap

Вместо вступления Я начну с плохого факта: рынок начинающих разработчиков сейчас слишком “перегрет” и для того, чтобы сейчас “войти в IT” нужно приложить усилий больше, чем 95% остальных. Звучит страшно, но из своего опыта я могу сказать следующее: многие не знаю что делать и в какую сторону копать, чтобы иметь это преимущество. Я попробую описать тот необходимый минимум, освоив который, вы будете выделяться на фоне конкурентов. Другой важный факт: знания языка программирования Java недостаточно для разработки хоть каких-то приложений.
